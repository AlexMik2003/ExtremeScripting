ExtremeXOS InSite Quick Start Guide

The ExtremeXOS InSite Quick Start Guide provides an introduction to the APIs. It describes the architecture and layout of the available APIs and gives an overview of how to use them. See the ExtremeXOS InSite Reference for a detailed description of the APIs.

ExtremeXOS InSite Reference

The ExtremeXOS InSite Reference Table provides a table of all of the WSDL and XSD files provided in this SDK, with links to a detailed description of the available APIs.

The ExtremeXOS InSite Reference List provides a detailed description of all the available APIs. It describes the interfaces and objects defined in the WSDL and XSD files.
