API Reference


Web Services Description Language (WSDL) files define the operations or interfaces that are available.

WSDL
Documentation
Description
event.wsdl
event
APIs that must be implemented to be able to catch XML formatted events generated by the switch.
switch.wsdl
switch
Switch APIs handle ports, VLANs, stacking, inventory, port statistics, user accounts, RADIUS management, TACACS management, and session management.
upm.wsdl
upm
UPM APIs handle profiles and timers.
xos.wsdl
xos
ExtremeXOS APIs handle VLANs, FDBs, dynamic ACLs, policies, configurations, and CLI commands.

XML Schema Definitions (XSD) files define the objects for the operations.

XSD Schema
Documentation
Description
aaa.xsd
aaa
Contains definitions of AAA Objects such as UserAccount, Session, and RadiusConfig.
common.xsd
common
Contains the definitions for the commonly used simple and complex types such as int, string, and boolean.
dhcp.xsd
dhcp
Contains complex types DHCPOptions and DHCPConfig.
fdb.xsd
fdb
Contains the definition for the MacTrackingEntry object.
ems.xsd
ems
Contains the definition of the EMS event object.
event.xsd
event
Contains the general definition of an event object.
idmgr.xsd
igmgr
Contains the definitions for the Identity Manager component, including UserIdentityData, IdentityManagerConfig, and KerberosSever objects.
ipfix.xsd
ipfix
Contains the definitions for the IPFIX component.
l2protocol.xsd
l2protocol
Contains the type definitions for AdminStates.
port.xsd
port
Contains definitions of Port Objects such as PortConfig, PortQoS, PortError, and PortUtilization.
snmp.xsd
snmp
Contains definitions of SNMP Objects such as GeneralConfig, SNMPStats, and SNMPCommunityEntry.
switch.xsd
switch
Contains definitions of interfaces such as get, set, create, and delete as well as the list of switch objects that can be accessed.
system.xsd
system
Contains definitions of Inventory Objects like Fan, PowerSupply, SlotInfo, SwitchInfo, and StackNode.
upm.xsd
upm
Contains declarations such as getProfileRequest, createProfileRequest, deleteProfileRequest.
vlan.xsd
vlan
Contains definitions of Vlan Objects such as VlanConfig, Protocol, and VirtualRouter.
