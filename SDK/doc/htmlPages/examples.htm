Examples

The ExtremeXOS InSite SDK contains example programs that show how the APIs can be used. The code snippets in these programs are meant to illustrate how you can perform similar operations in your application code.

Java Programming Examples

These example programs were written using Java SE 6 and Apache Axis 1.4. You may have to customize the code snippets in your application to suit your programming environment.

NOTE: Details on building and executing the example code is provided in README.txt
