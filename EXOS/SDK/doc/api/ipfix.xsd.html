IpfixPortData

SuperTypes common:ExosBase
SubTypes
The IPFIX Port parameters. ; Version : EXOS 12.5, Access : READ, WRITE, CREATE
Elements
common:StringLen8 portList
Port number. When referring to a single port the value is just that port number.

This is also used to specify a set of ports when setting the same configuration for multiple ports. In this case, the value is a comma separated list of ports or port ranges. For example, "1:1, 1:3, 2:1-2:3" means ports 1:1, 1:3, 2:1,2:2,2:3. ; Version : EXOS 12.5, Access : READ, WRITE, INDEX
FeatureStatusTypes ipfixPortEnabled
IPFIX State for the Port (or list). This must be specified. ; Version : EXOS 12.5, Access : READ, WRITE
FlowDirectionTypes ipfixFlowDirection
Flow Direction to Meter. This can be Ingress, Egress or Ingress-and-Egress. Default is Ingress only. ; Version : EXOS 12.5, Access : READ, WRITE
FlowTypes ipfixFlowType
Flow Type to Meter. IPv4, IPv6, NonIP (L2) or All. By default, all traffic types are metered. ; Version : EXOS 12.5, Access : READ, WRITE
RecordTypes ipfixRecordType
Dropped, Non-dropped, or Both. By default, all traffic is metered. ; Version : EXOS 12.5, Access : READ, WRITE
common:IPAddress ipfixFlowKeyIPv4SourceAddressMask
When the IPv4 Source Address is used as a Flow Key, this mask can control which bits are significant. This can be used to aggregate flows. For instance, you might use the subnet mask as the flow key mask to aggregate by subnet. ; Version : EXOS 12.5, Access : READ, WRITE
common:IPAddress ipfixFlowKeyIPv4DestinationAddressMask
When the IPv4 Destination Address is used as a Flow Key, this mask can control which bits are significant. This can be used to aggregate flows. For instance, you might use the subnet mask as the flow key mask to aggregate by subnet. ; Version : EXOS 12.5, Access : READ, WRITE
common:IPAddress ipfixFlowKeyIPv6SourceAddressMask
When the IPv6 Source Address is used as a Flow Key, this mask can control which bits are significant. This can be used to aggregate flows. For instance, you might use the subnet mask as the flow key mask to aggregate by subnet. ; Version : EXOS 12.5, Access : READ, WRITE
common:IPAddress ipfixFlowKeyIPv6DestinationAddressMask
When the IPv6 Destination Address is used as a Flow Key, this mask can control which bits are significant. This can be used to aggregate flows. For instance, you might use the subnet mask as the flow key mask to aggregate by subnet. ; Version : EXOS 12.5, Access : READ, WRITE
