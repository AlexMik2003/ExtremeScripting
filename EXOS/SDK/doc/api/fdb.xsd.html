MacTrackingEntry

SuperTypes common:ExosBase
SubTypes
MAC tracking entry configuration. This object is used to configure the fdb MAC tracking table on the switch. NOTE: There can be hundreds of entries configured on a switch. In these cases it is recommended that the paging mechanism available with getRequest be used when fetching all the entries from the switch. ; Version : EXOS 12.3, Access : READ, CREATE, DELETE
